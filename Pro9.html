High-Level Language


Un computador puede desarrollar aplicaciones, estas aplicaciones son un conjunto de instrucciones que le dicen al computador que hacer para idear una solución a un problema particular. Para el desarrollo de aplicaciones se usan lenguajes de programación como Java, c++, c entre otros. Hay dos tipos principales de lenguajes de programación:



•Lenguajes de bajo nivel:

Son aquellos en los que sus instrucciones ejercen un control directo sobre el hardware y están condicionados por la estructura física de las computadoras que lo soportan:
Ejemplos: Lenguaje de bajo nivel y lenguaje de ensamblador.

•Lenguajes de alto nivel

Se caracteriza por expresar los algoritmos de una manera adecuada a la capacidad cognitiva humana, en lugar de la capacidad con que los ejecutan las máquinas.



Para nuestra aplicación usamos un lenguaje llamado Jack, este es muy similar a Java, esta basado en objetos y no permite herencia.

Resumen de implementación:


Primero en la clase main es la función principal donde se ejecutan las demás, luego esta la función matrix que es una abstracción de un array multidimensional donde recibe un parámetro n y crea una matriz nxn de esta así como obtener los valores y asignar valores a el arreglo en la posición arr[j][i]=X , de esta manera se hace mucho más fácil la implementación.

La función mapa contiene el mapa del juego como el personaje, así como las cajas y posiciones donde se deben colocar para ganar el juego.

El mapa es una matriz nxn de esta manera recorrer el mapa es recorrer las posiciones de mi matriz. La función mapa cuenta además con validaciones propias del juego, así como los gráficos de este.

La clase Game es la clase que interactúa con el usuario y con el Mapa es el motor del juego dado el evento de que el jugador seleccione ciertas teclas, como arriba, abajo, derecha e izquierda, el otro lado activa el evento y de cumplir con las validaciones, se podrá mover o no.
